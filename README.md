# 📕 小红书 AI 自动发布矩阵系统 (Redbook AI Auto-Publisher)

这是一套专为个人及小型工作室打造的 **小红书全自动高质感图文/视频多模态分发中枢**。系统基于 Node.js 后端与 Playwright 无头浏览器模拟技术，并深度融合了当下最前沿的大视野视觉模型（如 Gemini 1.5 Pro, GPT-4o 等），实现从“素材洞察”到“爆款文生成”再到“自动上架”的零代码干预全流程闭环。\n\n## ✨ 核心特色功能\n\n*   **🤖 视听多模态 AI 前沿解析**\n    不只是单纯的文字续写！系统会像人类一样去“看”您传入的素材。无论是静态的绝美九宫格，还是动态的短视频，AI 将直接读取本地的无损二进制底层数据（或运用内置的 FFmpeg-static 进行关键帧切片技术），针对画面内发生的情境进行 100% 贴题的专业文案与话题提取。\n*   **🛡️ “拟态机”指纹级防封机制**\n    告别危险且容易被官方打击的非官方 API 黑客手段。本程序利用 Playwright 在本地浏览器拉起“创作者平台”，全程保留及复用您的本地登录 Session，一比一模拟真人的点击、拖拽和长传动作，绝对安全透明。\n*   **🕹️ 纯本地开箱即用的 GUI 控制台**\n    摆脱黑框敲命令的恐惧，自带极致优美的局部 GUI 网络服务器面板。随时编辑修改您的人设预设 Prompt 面板，或者随意更换电脑里的任意文件夹进行素材监听提取，一键点击，丝滑触达。\n*   **⚙️ 傻瓜式一键极速重置与部署环境**\n    自带 `一键部署与启动.bat` 强力脚本，无缝兼容所有 Windows 设备。无需懂 Node 或是 npm，双击即可完成依赖自动补齐、防冲突清扫及应用点亮的全自动化。\n\n## 🚀 极简破冰指南\n\n1.  **拉取源码并激活**\n    点击 GitHub 界面上的 `Code -> Download ZIP` 或是采用命令行克隆本仓库到您的 Windows 电脑。\n    解压后，双击根目录的 `一键部署与启动.bat` 脚本，静等服务自动安装所有依赖并在浏览器内弹出工作面板。\n2.  **对接您的 AI 大脑**\n    来到左侧【配置中心】，填写支持调用通用 OpenAI `v1/chat/completions` 标准接口规范的、并自带**高维视觉多模态处理能力**的基础模型服务（如 Gemini 等大代理渠道）。\n3.  **配置调性模型（可选）**\n    针对您的账号性质（如美妆博主、极客数码等），在左侧菜单编辑出个性十足的角色扮演**提示词模板**。\n4.  **指定灵感源**\n    进入主页面板。您可以只把几张图片放入一个空白的素材隔离文件夹；或者是**只把一个纯粹的短视频放到文件夹**里。\n    点击 `选择文件夹` 并定位到它。\n    如果是长视频，可在下方小黄框内设定要给 AI 切取多少张关键画面。\n5.  **开始降维打击**\n    点击 `▶ 开始智能生文`。后台将光速运转抽出您的核心论点。\n    确认大模型写出的内容对味后，点击发布。看着属于您的隐形员工帮您接管全网分发吧！\n\n---\n\n> **⚠️ 免责声明 (Disclaimer)**\n> \n> 本工具旨在协助用户提高工作效率和实现个人运营流水线的自动化，仅用于学习与交流自动化及大语言调用技术。\n> **请使用者务必遵守小红书的社群自律指引与平台公约**，切勿大量发布无意义垃圾信息（Spam）或侵权内容。滥用工具导致的账号限流及封禁等一切行为后果请使用者自行承担。
